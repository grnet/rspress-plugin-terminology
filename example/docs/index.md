# Welcome to Rspress Terminology Plugin

This is an example project demonstrating the **rspress-terminology** plugin.

## What is this plugin?

The rspress-terminology plugin enables you to:

- Define technical terms with markdown files
- Display hover tooltips when users mouse over term links
- Auto-generate a glossary page with all your terms
- Manage terminology from a single source of truth

## Try it out

Hover over these terms to see the tooltips:

- You need an [API Key](./terms/api-key) to authenticate with the API.
- The API implements [Rate Limiting](./terms/rate-limiting) to prevent abuse.
- Set up a [Webhook](./terms/webhook) to receive event notifications.
- We use [OAuth](./terms/oauth) for secure third-party authentication.

## Features

### ✨ Hover Tooltips

When you hover over a term link, a tooltip appears showing a brief definition. This helps readers understand technical terms without leaving the current page.

### 📚 Auto-Generated Glossary

The plugin automatically creates a glossary page that lists all defined terms. Check out the [Glossary](./glossary) page.

### 🎯 Smart Linking

Just use standard markdown links to your term files - the plugin handles the rest!

```markdown
Learn about [OAuth](./terms/oauth).
```
